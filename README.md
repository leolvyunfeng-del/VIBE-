# VIBE Russia Marketing 官网

VIBE 单页品牌官网，使用 React + Tailwind CSS 构建，面向潜在客户、品牌方和合作伙伴展示俄罗斯市场出海营销服务能力。

## 官网整体结构规划

1. Hero 首屏：品牌定位、核心价值、联系我们 / 查看服务 CTA、俄罗斯市场网络视觉。
2. 我们是谁：说明 VIBE 的定位、服务对象、服务范围和跨境协作价值。
3. 为什么选择俄罗斯市场：展示社媒活跃、电商增长、本土平台生态、中国品牌接受度提升等机会。
4. 我们的服务：六个服务卡片，覆盖 KOL、社媒、广告投放、线下广告、市场策略、PR传播。
5. 我们的方法论：市场洞察、策略制定、达人筛选、内容共创、投放执行、数据复盘。
6. 案例展示：三个占位案例，后续可替换为真实案例数据和图片。
7. 我们的优势：本地资源、平台理解、资源覆盖、中文协作、一体化服务。
8. 联系我们：静态需求表单和邮箱、微信、Telegram 占位联系方式。

## 页面模块文案

- Hero 标题：俄罗斯市场出海营销专家
- Hero 副标题：帮助中国品牌在俄罗斯建立认知、获取流量、沉淀信任。
- 我们是谁：VIBE 是一家专注俄罗斯市场的出海营销服务团队，为计划进入或正在深耕俄罗斯的中国品牌，提供从市场策略到营销执行的一体化支持。
- 市场机会：俄罗斯不是简单的流量市场，而是一个需要理解平台、语言、渠道与信任机制的本地化市场。
- 服务说明：以俄罗斯本土渠道和中文团队协作能力为基础，帮助品牌完成从策略、内容、传播到效果复盘的全链路执行。
- 方法论说明：VIBE 用流程化的方法管理跨境营销的不确定性，让每一次投放都能沉淀为下一轮增长的依据。
- 联系说明：告诉我们你的品牌、品类、目标和当前阶段，我们会基于俄罗斯市场给出初步沟通建议。

## 本地运行

```bash
pnpm install
pnpm dev
```

构建生产版本：

```bash
pnpm build
```

联系表单当前为静态展示，后续可接入邮件通知、CRM、企业微信机器人或表单服务。
